## 重要な注意事項

### データバックアップ
- **移行前に必ず完全バックアップを取得してください**
- 複数の場所にバックアップを保存してください

### 並行運用期間
- **1-2週間の並行運用期間を設けることを強く推奨します**
- この期間中に十分な検証を行ってください

### コスト管理
- **予算アラートを設定してください**
- 定期的にコストを見直してください

### セキュリティ
- **機密情報はAWS Secrets Managerを使用してください**
- セキュリティグループの設定を適切に行ってください
