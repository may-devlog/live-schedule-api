# テストユーザーの作成方法

テストユーザーを作成する方法を用意しました。

## 方法1: Rustコマンドを使用（推奨・最も簡単）

### 実行方法
```bash
cd backend
cargo run --bin create_user -- <メールアドレス> <パスワード>
```

### 例
```bash
cargo run --bin create_user -- test@example.com password123
```

この方法が最も簡単で確実です。パスワードのハッシュ化も自動で行われます。

## 方法2: メールアドレスを教えていただければ、こちらで作成します

メールアドレスとパスワードを教えていただければ、以下のコマンドで作成できます：

```bash
cd backend
cargo run --bin create_user -- <メールアドレス> <パスワード>
```

## 作成されるユーザーの特徴

- **メール確認済み** (`email_verified = 1`) として設定されます
- そのため、**すぐにログインできます**
- パスワードはbcryptでハッシュ化されて保存されます

## 注意事項

- 既に同じメールアドレスのユーザーが存在する場合は、エラーが表示されます
- パスワードは平文で保存されません（bcryptでハッシュ化されます）
